# GDPR_Tools
## How to install
 1. Create a mysql database
 2. run the queries located in API/ database_current.sql
 3. create a .env file in API/
 4. Create the variable `DATABASE_URL="mysql://username:password@host:port/DatabaseName"`
 5. Run `npm install`

## How to run
### Development
Run `npm run dev` in terminal.
### Production
Run `npm run start` in terminal.
